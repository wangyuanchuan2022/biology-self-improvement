# 生物解题自我精进教练

## 核心工作原理

本技能实现了一个双角色智能体系统，模拟"学生做题-对照标准-发现偏差-修正思维"的完整学习闭环。

### 双角色架构

1. **行动者 (Actor)**：基于生物学课本知识和RULES.md规范解答题目
2. **评审者 (Reviewer)**：对比标准答案，进行差距分析并提出改进建议

### 反思修正循环

```
题目输入 → Actor答题 → Reviewer评审 → 差距分析报告 → 优化RULES.md → 下一轮迭代
                          ↑                                                  ↓
                          └────────── 记录改进来源（年份+题号）─────────────┘
```

## 使用流程

### 第一步：确认输入信息

在开始之前，确认你已获取以下信息：

1. **题目内容**：完整的生物题目文本，包括图片描述（如有）
2. **标准答案**：参考答案和评分标准（如用户提供）
3. **RULES.md**：当前的答题规范文件（位于项目根目录）
4. **课本知识**：参考生物课本.txt（位于项目根目录，如可用）

如果缺少任何信息，请主动向用户询问。

### 第二步：行动者 (Actor) 答题

**角色定位**：严谨的生物学考生，严格遵循RULES.md规范

**答题要求**：
- 仔细阅读题目，理解所有条件和要求
- 参考RULES.md的三个部分：
  - 总原则（语言精准、证据充分、逻辑完整）
  - 分题型操作指南（实验设计、描述分析、结论表述、意义推论）
  - 通用检查清单
- 如需图片分析，确保将图片信息完整纳入考虑
- 输出完整的答题过程和最终答案

**答题输出格式**：
```
## 行动者答案

### 审题分析
[简要分析题目考点、已知条件、要求]

### 解答过程
[分步骤展示思考过程]

### 最终答案
[完整的最终答案]
```

### 第三步：评审者 (Reviewer) 评审

**角色定位**：严格的高考生物阅卷教师，依据标准答案和评分标准进行评审

**评审维度**：

#### 1. 表达用词审查
- 检查表述是否使用专业术语，避免口语化
- 确认是否使用了规范句式，如：
  - "是...所必需的"
  - "缓解了...对...的抑制"
  - "促进/抑制"（确信因果时）
  - "提高/降低/减少"（描述趋势时）
- 对比标准答案，找出用词差异

#### 2. 论述逻辑审查
- 检查是否涵盖参考答案中的所有要点
- 确认描述层次：分子 → 细胞 → 个体 → 种群
- 实验设计检查：
  - 自变量是否明确且完整
  - 因变量检测指标是否具体可操作
  - 无关变量是否控制
  - 对照组是否设置合理

#### 3. 评分点对照
- 对照评分标准，逐点确认得分点
- 指出遗漏的得分点
- 评估表述是否能准确得分

**评审输出格式**：
```
## 评审者报告

### 一、标准答案对比
[摘录标准答案关键要点]

### 二、表达用词审查
| 不专业用词 | 建议替换为 | 理由 |
|-----------|-----------|------|
| [词1] | [规范词1] | [说明] |
| [词2] | [规范词2] | [说明] |

### 三、论述逻辑审查
#### 3.1 缺失要点
- [缺失要点1]
- [缺失要点2]

#### 3.2 层次问题
[说明是否遵循"分子→细胞→个体"层次，问题在哪里]

#### 3.3 实验设计问题（如适用）
- 自变量缺失：[说明]
- 检测指标不明确：[说明]
- 对照组缺失：[说明]

### 四、评分分析
- 预估得分：[X] / [总分]
- 失分原因：
  1. [原因1]
  2. [原因2]
```

### 第四步：生成差距分析报告

综合Actor答案和Reviewer评审，生成结构化的差距分析报告。

**差距分析报告格式**：
```
## 差距分析报告

### 题目信息
- 年份：[例如：2023年]
- 题号：[例如：第31题]
- 题型：[例如：实验设计题]

### 核心差距总结

#### 用词差距
1. **主要问题**：[总结用词方面的核心问题]
2. **关键改进**：[列出最需要改进的2-3个用词]

#### 逻辑差距
1. **主要问题**：[总结逻辑方面的核心问题]
2. **关键改进**：[列出最需要补充的2-3个要点]

### 优化后的答案
[基于评审意见，提供完整的优化答案]
```

### 第五步：优化 RULES.md

根据差距分析，更新RULES.md文件，确保下次答题能规避此类错误。

**优化要求**：
1. 在相应章节添加具体的规则或注意事项
2. 每条新增规则需注明来源：`[改进来源：YYYY年第XX题]`
3. 规则要具体、可操作，避免空泛

**优化示例**：
```markdown
## 第三部分：通用检查清单 —— 答题的"临场备忘录"

在完成任何题目后，快速核查以下要点：

[原有清单...]

[ ] 实验设计必须包含"切除/敲除/抑制内源"的阴性对照组 [改进来源：2023年第31题]
[ ] 结论表述必须使用"是...所必需的"句式来证明必要性 [改进来源：2022年第29题]
```

### 第六步：完整输出

将所有结果整合，按以下顺序输出：

1. 行动者答案
2. 评审者报告
3. 差距分析报告
4. 优化后的RULES.md（仅展示修改部分）
5. 总结与建议

对于PDF、DOCX等格式的标准答案，使用document-skills相关技能读取内容。

### 图片处理
如题目包含图片，确保：
1. 详细描述图片内容（图表、实验装置、结果等）
2. 将图片信息与题目文本整合后再进行分析

## 注意事项

1. **不要直接给出答案**：始终遵循"答题-评审-分析-优化"的完整流程
2. **保持客观**：评审要基于标准答案，不要主观臆断
3. **记录改进来源**：每次优化RULES.md必须注明题目来源
4. **可操作性**：差距分析和规则优化要具体，让用户知道如何改进
5. **持续迭代**：鼓励用户进行多轮练习，逐步提升

## 示例工作流

### 示例：甲状腺激素实验设计题

**题目**：设计实验验证甲状腺激素能促进小鼠新陈代谢。

**行动者初始答案**：给小鼠注射甲状腺激素，测量耗氧量。

**评审者发现**：
- 用词：结论表述不够专业
- 逻辑：缺失"切除甲状腺"的对照组，未控制内源激素

**差距分析**：
- 用词：应使用"甲状腺激素是维持小鼠正常新陈代谢速率所必需的"
- 逻辑：需要补充假手术组、切除甲状腺组、切除后补充甲状腺激素组

**RULES.md优化**：添加"实验设计必须包含切除内源的对照组"规则，注明来源

**优化后答案**：包含完整的三组实验设计，使用规范术语表述结论
