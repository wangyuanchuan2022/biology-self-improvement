# 生物解题自我精进教练 - 使用示例

## 示例1：甲状腺激素实验设计题

### 题目
设计实验验证甲状腺激素能促进小鼠新陈代谢。

### 运行方式
将以下内容复制到对话中：

```
请使用biology-self-improvement-tutor技能解答这道题：

题目：设计实验验证甲状腺激素能促进小鼠新陈代谢。

标准答案：
1. 实验分组：
   甲组：切除甲状腺 + 注射生理盐水
   乙组：切除甲状腺 + 注射甲状腺激素
   丙组：假手术 + 注射生理盐水
2. 检测指标：测量各组小鼠的耗氧量
3. 预期结果：甲组耗氧量降低，乙组恢复正常，丙组正常
4. 结论：甲状腺激素是维持小鼠正常新陈代谢速率所必需的。

题目来源：2023年第31题，实验设计题
```

## 示例2：基因表达分析题

### 题目
分析下图中基因表达量的变化，并说明其生物学意义。
（图描述：X轴为药物浓度，Y轴为mRNA相对含量；X基因曲线：先升后降；Y基因曲线：持续下降）

### 运行方式
将以下内容复制到对话中：

```
请使用biology-self-improvement-tutor技能解答这道题：

题目：分析下图中基因表达量的变化，并说明其生物学意义。
图片描述：X轴为药物浓度（从低到高），Y轴为mRNA相对含量；X基因的曲线：在低浓度时升高，高浓度时下降；Y基因的曲线：随药物浓度增加持续降低。

标准答案：
分子水平：X基因mRNA含量在低浓度药物下升高，高浓度下下降；Y基因mRNA含量随药物浓度增加持续降低。
细胞水平：X基因可能参与应激反应，Y基因可能参与细胞增殖。
个体水平：低浓度药物可能促进细胞适应，高浓度可能产生毒性。
意义：在药物开发中，需关注剂量效应，避免过高浓度产生副作用；在临床上，需合理控制药物用量以达到最佳治疗效果。

题目来源：2022年第29题，描述分析题
```
